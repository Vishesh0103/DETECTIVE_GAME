# 🕵️‍♂️ Gemini AI Murder Mystery

A browser-based detective game where every case is **freshly generated by Google Gemini AI** — unique suspects, alibis, crime scenes, and hidden contradictions every time you play.

Built with **Streamlit** + **Gemini 2.5 Flash**.

---

## 🎮 How to Play

1. **Interrogate** the 3 suspects and read their alibi statements
2. **Search** the crime scene areas for physical evidence
3. **Expose** the contradiction — find which clue breaks the killer's alibi
4. **Accuse** the murderer before you run out of confidence

Every correct action earns points. Wrong accusations and false contradictions cost you.

---

## ✨ Features

- 🤖 **AI-generated cases** — no two games are the same
- 🔍 **3-step deduction loop** — interrogate → investigate → expose
- 🧠 **Contradiction system** — logic-based alibi breaking mechanic
- 🏆 **Score tracker** — rewards sharp thinking, penalizes guessing
- 🎨 **Noir detective UI** — dark theme with classic detective aesthetics
